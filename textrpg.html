TextRPG

2024 Game

About This Project

TextRPG is a text-based role-playing game developed for the Windows platform. In this project, players take on the role of a villager who must defend their village from monster attacks through narrative-driven choices and simple strategic text-based combat. The game focuses on interactive storytelling, player decision-making, and progression through branching scenarios. This project involved game logic design, narrative development, and implementation of text-based gameplay mechanics.

Key Features

  • Text-Based RPG Gameplay
  • Narrative-Driven Choices
  • Simple Combat System
